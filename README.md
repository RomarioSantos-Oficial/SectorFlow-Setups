### 📥 Download e Instalação Rápida (Recomendado)

> Não precisa instalar Python. Baixe e execute diretamente!

#### Passo 1 — Baixar o executável

**👉 [Clique aqui para baixar o SectorFlow Setups v1.0-beta](https://github.com/RomarioSantos-Oficial/SectorFlow-Setups/releases/tag/v1.0-beta)**

1. Acesse o link acima
2. Role até a seção **Assets**
3. Clique em `SectorFlowSetups.exe` para baixar

#### Passo 2 — Requisitos antes de executar

| Requisito | Detalhe |
|---|---|
| 🖥️ Sistema Operacional | Windows 10 ou 11 (64-bit) |
| 🎮 Jogo | **Le Mans Ultimate instalado e rodando** |
| 📁 Arquivo base | Um arquivo `.svm` de setup do LMU |

> ⚠️ **IMPORTANTE:** Esta ferramenta funciona **exclusivamente com Le Mans Ultimate**. Não é compatível com outros simuladores.

1. Localize o arquivo `SectorFlowSetups.exe` baixado

3. O aplicativo abrirá automaticamente

#### Passo 4 — Primeiro uso

```
1. Abra o Le Mans Ultimate
2. Aguarde o indicador LMU ficar VERDE no app
3. Vá para a aba "Setup"
4. Clique em "Carregar .svm" e escolha seu setup base
5. Entre na pista e complete pelo menos 1 volta
6. Volte ao app → clique em "Sugestões IA" ou "Heurísticas"
7. Aplique as sugestões e teste na pista!
```

#### Regras de Uso

- ✅ Apenas para **Le Mans Ultimate**
- ✅ Jogo deve estar **aberto e rodando** para conectar
- ✅ Precisa de um arquivo `.svm` como base de setup
- ❌ Não funciona com rFactor 2, ACC ou outros sims
- ❌ Não funciona sem o jogo instalado (modo telemetria limitado)

#### Solução de Problemas Rápida

<details>
<summary>🔴 O app abre mas não conecta ao jogo</summary>

1. Confirme que o **Le Mans Ultimate está aberto**
2. Verifique se a **Shared Memory** está ativada nas configurações do LMU
3. Tente reiniciar o aplicativo com o jogo já aberto
4. Veja os logs em `%APPDATA%\SectorFlowSetups\logs\`

</details>

<details>
<summary>🔴 O app não sugere nada</summary>

1. Complete **pelo menos 1 volta completa** na pista
2. Confirme que carregou um arquivo `.svm` de base
3. Verifique se a aba **Telemetria** exibe dados ao vivo (números mudando)

</details>

<details>
<summary>🔴 A IA parece fraca no início</summary>

Isso é **normal e esperado**. No início, o sistema depende mais de heurísticas e vai melhorando conforme você completa voltas e envia feedback. Com mais dados, as sugestões ficam mais precisas.

</details>

<details>
<summary>🔴 Windows bloqueou o .exe</summary>

1. Clique com o botão direito no `SectorFlowSetups.exe`
2. Selecione **Propriedades**
3. Na parte inferior, marque **"Desbloquear"**
4. Clique em **OK** e execute novamente

</details>

| Requirement | Details |

```
1. Open Le Mans Ultimate
2. Wait for the LMU indicator to turn GREEN in the app
3. Go to the "Setup" tab
4. Click "Load .svm" and choose your base setup file

6. Return to the app → click "AI Suggestions" or "Heuristics"
7. Apply the suggestions and test on track!
```

#### Usage Rules

- ✅ Only for **Le Mans Ultimate**
- ✅ Game must be **open and running** to connect
- ✅ Requires an `.svm` file as setup base
- ❌ Does not work with rFactor 2, ACC or other sims
- ❌ Does not work without the game installed (limited telemetry mode)

#### Quick Troubleshooting

<details>
<summary>🔴 The app opens but won't connect to the game</summary>

1. Confirm **Le Mans Ultimate is open**
2. Check that **Shared Memory** is enabled in LMU settings
3. Try restarting the app with the game already running
4. Check logs at `%APPDATA%\SectorFlowSetups\logs\`

</details>

<details>
<summary>🔴 The app doesn't suggest anything</summary>

1. Complete **at least 1 full lap** on track
2. Confirm you loaded a base `.svm` file
3. Check that the **Telemetry** tab shows live data (numbers changing)

</details>

<details>
<summary>🔴 The AI seems weak at the start</summary>

This is **normal and expected**. At the beginning, the system relies more on heuristics and improves as you complete laps and provide feedback. With more data, the suggestions become more accurate.

</details>

<details>
<summary>🔴 Windows blocked the .exe</summary>

1. Right-click on `SectorFlowSetups.exe`
2. Select **Properties**
3. At the bottom, check **"Unblock"**
4. Click **OK** and run again

</details>

### Quick Start (Recommended)

**Option 1 - PowerShell:**
```PowerShell
.\start.ps1
```

**Option 2 - Command Prompt:**
```batch
start.bat
```

The script automatically:
- Checks Python 3.10+ is installed
- Creates virtual environment
- Installs PyTorch (CPU version)
- Installs all dependencies
- Starts the application

### Manual Installation

### 1. Download the project

Download or clone this repository.

### 2. Create a virtual environment

In PowerShell, within the project folder:

```PowerShell
python -m venv.venv
.\.venv\Scripts\Activate.ps1
```

### 3. Install dependencies

```PowerShell
pip install -r requirements.txt
```

### 4. Run the application

```PowerShell
python main.py
```

### Build Executable

To compile a standalone .exe (runs without Python):

```PowerShell
.\scripts\build.ps1
```

The executable will be created at `dist/SectorFlowSetups.exe`.

For full build and installer instructions, see [docs/INSTALL.md](docs/INSTALL.md).
